# App de Gastos (Flutter) 💸

App para registrar y llevar el control de tus gastos rápidos con una linda interfaz.

## 🚀 Cómo correr el proyecto en tu PC

Para probar la app localmente en tu computadora, tenés tres opciones súper rápidas:

**Opción 1: En el emulador Android**
Abrí una terminal en esta carpeta y ejecutá:
```bash
flutter run -d emulator-5554
```

**Opción 2: En tu navegador**
Abrí una terminal en esta carpeta y ejecutá:
```bash
flutter run -d edge
```
*(Se abrirá una ventana. Apretá `F12` y luego `Ctrl + Shift + M` para darle formato de celular).*

**Opción 3: Como una app nativa de Windows**
Si preferís que se abra como un programa de computadora normal:
```bash
flutter run -d windows
```

> **Tip:** Estando en la terminal, si hacés cambios en el código podés apretar la letra `r` para que se actualicen al instante en pantalla ("Hot Reload").
