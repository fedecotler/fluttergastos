# App de Gastos (Flutter) 💸

Una aplicación sencilla, elegante y rápida para registrar y llevar el control de tus gastos diarios, creada en Flutter.

## 🚀 Cómo correr el proyecto fácilmente

Este proyecto está construido con [Flutter](https://flutter.dev), por lo que podés correrlo tanto en tu celular, en un emulador o directamente en el navegador.

### Opción 1: Correr en tu celular (o emulador)
Si querés ver cómo funciona como una App Móvil tradicional (como Android):

1. **Abrí una terminal** en la carpeta principal del proyecto.
2. Asegurate de tener tu celular conectado por cable o un emulador encendido (lo podés verificar escribiendo `flutter devices`).
3. Ejecutá el siguiente comando:
   ```bash
   flutter run
   ```
   *(Si tenés más de un dispositivo, te va a preguntar en cuál de todos querés correrlo. Solo tenés que elegir el número).*

### Opción 2: Correr directamente en el navegador de tu compu (Edge / Chrome)
Es la manera as rápida de ver la aplicación mientras la editás, sin necesitar un emulador pesado de Android de fondo:

1. Abrí la terminal y ejecutá este comando:
   ```bash
   flutter run -d edge
   ```
   *(Podés cambiar `edge` por `chrome` si usás Google Chrome).*
2. Andá a la ventana que se abre y apretá `F12` (herramientas de desarrollador), y luego el atajo `Ctrl + Shift + M`. Esto pondrá el navegador **en modo Teléfono** para que puedas interactuar arrastrando y simulando los dedos en la pantalla.

---

## 💻 Comandos útiles

- `r`: Estando en la terminal y con el proyecto corriendo, apretás la tecla `r` para hacer un **Hot Reload** (recargar los colores o texto que cambies al instante, sin cerrar la app).
- `q`: Para salir y cerrar la aplicación (Quit).
